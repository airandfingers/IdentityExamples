Example 3.1: Tabstops and autofocus

More information on making your site accessible: http://www.w3.org/WAI/intro/accessibility.php

Please sign in

  • Add <label> tags. Not all browsers support placeholder text, and often people accidentally type before they read the textbox. Plus, screen readers make use of them.
  • Add "autofocus" to the username field so the cursor begins here
  • Add tabindex to username, password, checkbox, and submit buttons
  • Make sure that the button is type "submit" so people can hit return to log in.

Sample code

<label for="username">Email</label>
<input type="text" class="form-control" placeholder="Email address" name="username" required autofocus tabindex="1">
<button class="btn btn-lg btn-primary btn-block" type="submit" tabindex="4">Sign in</button>
