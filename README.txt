These examples are designed to accompany the Environments for Humans: JavaScript Summit 2013 ( http://environmentsforhumans.com/2013/javascript-summit/ ).

To get started, you need to do three things:

1. Install dependencies
> npm install

2. (Optional, if you want to use the third-party providers)
> cp keys.example.js keys.js
> $FAV_EDITOR keys.js
> Add in your github, facebook, and recaptcha keys

3. Run
> node app.js

Now the examples will be working for you at http://localhost:3000
